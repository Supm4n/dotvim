# Vim config 

.vimrc file and vim plugins I use

## Installation

`cd ~
git clone http://github.com/supm4n/dotvim.git ~/.vim
ln -s ~/.vim/.vimrc ~/.vimrc
ln -s ~/.vim/.vimrc ~/.gvimrc
cd ~/.vim
git submodule init
git submodule update`

Enjoy !
